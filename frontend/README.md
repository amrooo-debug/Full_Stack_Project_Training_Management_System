# Frontend (React + TypeScript + Vite)

This folder contains the frontend for the Training Management System.

The full project documentation — features, tech stack, local setup, environment
variables, running tests, and screenshots — lives in the **main README at the
repository root**:

- [`../README.md`](../README.md)

Quick reference (run these from this `frontend` folder):

```bash
npm install       # install dependencies (first time only)
npm run dev       # start the Vite dev server on http://localhost:5173
npm test          # run Vitest unit tests
npm run test:e2e  # run Playwright E2E tests (backend + frontend must be running)
npm run build     # TypeScript build + Vite production build
```
